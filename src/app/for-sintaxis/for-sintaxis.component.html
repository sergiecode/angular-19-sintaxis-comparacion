Ejemplo de FOR

Sintaxis Antigua (*ngFor)

<ul>
  <li *ngFor="let fruta of frutas; let i = index">
{{'    {{i+1}} - {{fruta}}'}}
  </li>
</ul>
  • {{ i + 1 }} - {{ fruta }}
Nueva Sintaxis ({{ "@for" }})

<ul>
  {{'@for (fruta of frutas; track $index; let i = $index) {'}}
    <li>{{'{{i+1}}'}} - {{'{{fruta}}'}}</li>
  {{'}'}}
</ul>
    @for (fruta of frutas; track $index; let i = $index) {
  • {{ i + 1 }} - {{ fruta }}
    • }
