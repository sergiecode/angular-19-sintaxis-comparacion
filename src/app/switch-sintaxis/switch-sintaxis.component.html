Ejemplo de SWITCH

Sintaxis Antigua ([ngSwitch])

<div [ngSwitch]="opcion">
  <div *ngSwitchCase="'A'">Has seleccionado A</div>
  <div *ngSwitchCase="'B'">Has seleccionado B</div>
  <div *ngSwitchDefault>Has seleccionado otra opción</div>
</div>
Has seleccionado A
Has seleccionado B
Has seleccionado otra opción
Nueva Sintaxis ({{ "@switch" }})

{{'@switch (opcion) {'}}
  {{'@case ("A") {'}}
    <div>Has seleccionado A</div>
  {{'}'}}
  {{'@case ("B") {'}}
    <div>Has seleccionado B</div>
  {{'}'}}
  {{'@default {'}}
    <div>Has seleccionado otra opción</div>
  {{'}'}}
{{'}'}}
@switch (opcion) { @case ('A') {
Has seleccionado A
} @case ('B') {
Has seleccionado B
} @default {
Has seleccionado otra opción
} }
